# Google Apps Script (GAS) バックエンド 導入＆設定ガイド

スマホやパソコンのブラウザから GitHub リポジトリへ直接ファイルをプッシュし、自動で動画生成（GitHub Actions）を起動・監視するために必要な GAS の設定マニュアルです。所要時間は約3〜5分です。

---

## 📋 手順 1: GAS プロジェクトの新規作成
1. Google ドライブ または [Google Apps Script (`https://script.google.com`)](https://script.google.com) を開きます。
2. 左上の「**＋ 新しいプロジェクト**」をクリックします。
3. プロジェクト名を「`爆モテ動画制作コントローラー GAS API`」に変更します。
4. デフォルトの `コード.gs` に、本プロジェクトの `gas/Code.gs` のコード全体をコピー＆ペーストして保存 (`Ctrl + S` または `Cmd + S`) します。

> **※スプレッドシート連携（オプション）**:
> 履歴ログをスプレッドシートに自動記録したい場合は、Google スプレッドシートを新規作成し、メニューの「**拡張機能 -> Apps Script**」からスクリプトエディタを開いて `Code.gs` を貼り付けてください。自動で `生成履歴ログ` シートに履歴が書き込まれます。

---

## 🔑 手順 2: GitHub Personal Access Token (PAT) の取得と設定
リポジトリへファイルをプッシュし、ワークフローを操作するためにトークンが必要です。

### ① GitHub トークンを作成
1. GitHub 右上アイコン -> **Settings -> Developer settings -> Personal access tokens -> Tokens (classic)** を開きます。
2. 「**Generate new token (classic)**」をクリック。
3. Note: `Bakumote Controller GAS API` など分かりやすい名前を入力。
4. Select scopes で **`repo`** (全チェック) と **`workflow`** にチェックを入れます。
5. 一番下の「**Generate token**」を押し、表示された `ghp_xxxxxxxxxxxxxxxxxxxx` から始まるトークンをコピーします。

### ② GAS のスクリプトプロパティに設定
1. GAS エディタの左側メニューから「**プロジェクトの設定 (歯車アイコン)**」をクリックします。
2. 一番下の「**スクリプト プロパティ -> スクリプト プロパティを追加**」をクリックします。
3. 以下の2項目を設定します：
   - プロパティ名: `GITHUB_TOKEN` / 値: コピーした GitHub トークン (`ghp_xxx...`)
   - プロパティ名: `REPO_NAME` / 値: `PoiTaro/tiktoker` (お使いのリポジトリ名)
4. 「**スクリプト プロパティを保存**」をクリック。

---

## 🚀 手順 3: Web アプリとしてデプロイ＆ URL の取得
1. GAS エディタ右上の青い「**デプロイ**」ボタン ->「**新しいデプロイ**」をクリック。
2. 左側の種類の選択 (歯車アイコン) で「**ウェブアプリ**」を選択します。
3. 次の設定を行います：
   - **説明**: `Bakumote Video Controller API v1`
   - **次のユーザーとして実行**: `自分 (お使いのGoogleアカウント)`
   - **アクセスできるユーザー**: `全員` (スマホの Web UI からパスワード認証なく AJAX アクセスするために「全員」に設定します)
4. 右下の「**デプロイ**」ボタンを押します。
   ※初回デプロイ時は「アクセスを承認」が求められます。「アカウントを選択」->「詳細」->「爆モテ動画制作コントローラー GAS API（安全ではないページ）に移動」->「許可」を押してください。
5. 発行された **「ウェブアプリの URL (`https://script.google.com/macros/s/XXXXX/exec`)」** をコピーします！

---

## 📱 手順 4: スマホ Web UI に URL を設定
`webui/index.html` をスマホやパソコンのブラウザで開いた際、画面上部の **「⚙️ GAS バックエンド設定」** 欄にコピーした **「ウェブアプリの URL」** を貼り付けて保存してください。

これで、スマホからの全操作がリポジトリとリアルタイム連動します！
